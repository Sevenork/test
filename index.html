Что такое GitHub Pages?

GitHub Pages — это способ размещения веб-страниц прямо из вашего репозитория на GitHub.

Как использовать GitHub Pages?

Вы можете создать новую ветку с названием gh-pages и разместить свои файлы там для публикации.
